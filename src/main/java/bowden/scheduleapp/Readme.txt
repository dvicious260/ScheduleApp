IDE, JDK, and JavaFX Versions:
IDE: IntelliJ IDEA Community 2022.1
JDK: Oracle OpenJDK 18.0.2.1
JavaFX: JavaFX-SDK-18.0.2

Directions for Running the Program:

Open the project in IntelliJ IDEA.
Ensure that the project SDK is set to Java SE 18.0.2.1 and that the JavaFX library is included in the project.
Build and run the application by clicking the "Run" button in the IDE. Ensure that you have valid login credentials for the login screen.
Description of Additional Report:
The additional report I ran in part A3f was a listing customers by country. It provides a tableview which shows each country and how many of the customers in the database are from that country.
